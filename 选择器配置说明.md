# 选择器配置说明

本工具优先使用 `selectors.json`，若不存在或解析失败则回退到 `config.json` 中的 `selectors`。

## 基本格式

`selectors.json` 采用如下结构：

```json
{
  "version": "1.1.0",
  "lastUpdated": "2026-07-20",
  "selectors": {
    "tweet": {
      "primary": "[data-testid='tweet']",
      "fallback": "article[role='article']"
    }
  }
}
```

## 更新流程

1. 打开 https://x.com 页面，按 `F12` 进入开发者工具
2. 找到目标元素并复制选择器
3. 更新 `selectors.json` 的 `primary` 或 `fallback`（`primary` 优先写与界面语言无关的 `data-testid`）
4. 运行 `npm run test:selectors` 验证格式

> 注意：`primary` 和 `fallback` 会被合并成一条逗号分隔的选择器列表，命中顺序由 DOM 位置决定，不是由书写顺序决定。备用选择器必须与主选择器语义一致。

更详细说明请查看 `docs/SELECTOR_UPDATE_GUIDE.md`。
