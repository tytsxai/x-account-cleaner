# 选择器配置说明

本工具优先使用 `selectors.json`，若不存在或解析失败则回退到 `config.json` 中的 `selectors`。

## 基本格式

`selectors.json` 采用如下结构：

```json
{
  "version": "1.0.0",
  "lastUpdated": "2025-01-01",
  "selectors": {
    "tweet": {
      "primary": "[data-testid='tweet']",
      "fallback": "article[role='article']"
    }
  }
}
```

## 更新流程

1. 打开 Twitter 页面，按 `F12` 进入开发者工具
2. 找到目标元素并复制选择器
3. 更新 `selectors.json` 的 `primary` 或 `fallback`
4. 运行 `npm run test:selectors` 验证格式

更详细说明请查看 `docs/SELECTOR_UPDATE_GUIDE.md`。
