# 启动指南

本指南说明如何以稳定方式运行工具，并提供最小化的生产运行建议。

## 本地/日常使用

1. 安装依赖：`install.sh` / `install.bat`
2. 配置 `config.json`
3. 启动：`start.sh` / `start.bat` 或 `npm run build && npm run start:prod`

## 生产/长期运行（推荐）

1. 安装依赖：`npm install`
2. 安装浏览器：`npx playwright install chromium`
3. 构建：`npm run build`
4. 启动：`npm run start:prod`

## 运行前检查

- `config.json` 配置正确
- 如使用 `selectors.json`，先运行 `npm run test:selectors`
- `.env` 为可选，未配置默认走手动登录

## 日志与运行摘要

- 运行日志：`logs/combined.log`、`logs/error.log`
- 运行摘要：`logs/run-summary-<runId>.json`

## 停止与退出

按 `Ctrl+C` 可安全退出，程序会尝试关闭浏览器并写入运行摘要。
