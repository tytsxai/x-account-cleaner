# 登录说明

本工具支持两种登录方式：手动登录（默认）与自动登录（可选）。

## 方式一：手动登录（推荐）

1. 不需要配置 `.env`
2. 启动后会打开浏览器并进入登录页
3. 在浏览器内完成登录与可能的安全验证
4. 登录成功后，程序自动继续执行

## 方式二：自动登录（可选）

1. 复制 `env.example` 为 `.env`
2. 填写账号信息：
   - `TWITTER_USERNAME`
   - `TWITTER_PASSWORD`
3. 运行程序后将自动登录

## 登录状态保存

登录状态会保存在 `browser-data/state.json`。如需重新登录，删除该文件后再运行。

## 安全提示

- `.env` 包含敏感信息，请勿分享或提交到仓库
- 若开启了 2FA，仍可能需要手动输入验证码
- 使用 `HEADLESS=true` 时，需提供账号密码或已保存登录状态
