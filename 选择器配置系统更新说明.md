# 选择器配置系统更新说明

本项目支持独立的选择器配置文件 `selectors.json`，用于在 X（原 Twitter）页面结构变化时快速修复。

## 优先级

1. `selectors.json`（优先）
2. `config.json` 中的 `selectors`（回退）

## 建议更新规则

- 更新选择器后同步修改 `selectors.json` 的 `version` 与 `lastUpdated`
- 变更后执行 `npm run test:selectors` 验证格式
- 备用选择器不要匹配语义不同的按钮：合并后的列表按 DOM 顺序命中，而非书写顺序
- 若仍失败，请参考 `docs/SELECTOR_UPDATE_GUIDE.md`
- 验证可用的新选择器欢迎提 Issue 或 PR 回馈项目
